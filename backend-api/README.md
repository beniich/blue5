# Backend API partagé pour Dash1cc

API backend en TypeScript avec Express pour School 1cc et CRM Pro.cc

## Démarrage rapide

\\\ash
# Installation
npm install

# Configuration
cp .env.example .env
# Éditer .env avec vos vraies valeurs

# Prisma
npx prisma generate
npx prisma migrate dev

# Développement
npm run dev

# Production
npm run build
npm start
\\\

## Ports
- API: http://localhost:3000
- Prisma Studio: http://localhost:5555
